# stm32-blinkLR2
Лабораторна робота 2. Програмування портів вводу-виводу загального 
призначення на основі прототипу операційної системи нескінченного циклу опитування 
реального часу.
Обладнання: Налагоджувальна плата STM32F103C8T6, ST-LINK V2 SWIM програматор STM8 STM32, CubeIDE
Завдання: 
1. Запрограмувати світлодіод  PD12 та PD13  з частотою повторення імпульсів 1 Гц,
тривалість імпульсу 50 мс. Зчитати стан кнопки PA0 та відобразити його на світлодіоді PD13
(натиснута — світить постійно).
